# ✨ Magic and Useful Items

This chapter collects practical magic-related tips that provide concrete advantages without replacing complete spell lists.

Referenced spells and magical effects are summarized in [[15_Appendix#^glossary-spells|Glossary: Spells, Buffs, and Magical Effects]].

> [!tip] Core habit
> Spend spell resources deliberately: learn scrolls that fill a real gap, support important skill checks, and check missable magic-item areas before leaving them.

## 📖 Learning Wizard Spells ^wizard-learning

**Markers:** ⭐⭐ 💰 ✨

Open the spellbook with `K`. **Learn More Spells** shows unknown spells, costs, and minimum level.

Evocation spells are cheaper for Evocation Wizards. Keep this in mind before learning expensive spells.

### 🎯 Practical Use

| Step | Benefit |
|---|---|
| Open the spellbook with `K` | Quickly check available options |
| Use Learn More Spells | See unknown spells, costs, and minimum level |
| Compare costs | Spend gold and scrolls more deliberately |
| Check Evocation Wizards | Evocation spells are cheaper and marked accordingly |

Not every scroll needs to be learned immediately. Before expensive learning, check current role, minimum level, and gold reserves.

For Evocation Wizards, Evocation scrolls can cost half as much to learn. This is visible in the learning interface and should be checked before spending gold.

### ✅ Wizard Learning Routine

Use a consistent routine when handling scrolls:

1. Open the spellbook with `K`.
2. Use **Learn More Spells**.
3. Check which spells are unknown.
4. Compare cost, minimum level, and practical use.
5. Learn spells that fill an actual party need first.

This prevents spending gold on spells that are technically available but not useful for the current party.

## 🔁 Swap Prepared Spells Outside Combat ^prepared-spells

**Markers:** ⭐⭐⭐ ⏱️ ✨

Prepared casters can change prepared spells outside combat. Do this before hard fights, exploration puzzles, or important dialogue instead of carrying the same spell list all day.

Practical examples:

- prepare utility spells before exploration,
- prepare control and defensive spells before a boss,
- swap in Speak with Dead, Feather Fall, or Enhance Ability when the next section clearly needs them,
- keep scrolls for situations where swapping is impossible or the caster is absent.

Primary source video: [16 Secret Tips You STILL Aren't Doing In Baldur's Gate 3 - Doms Roundtable](https://www.youtube.com/watch?v=fF28SqTbOOQ). Additional source videos: [BG3 Guide: Learn Wizard Spells from Scrolls - Chupacabra Tutorials](https://www.youtube.com/watch?v=6lGNMao5qkY), [I Wish I Knew These Tips Before Starting Baldur's Gate 3 - Jolsn](https://www.youtube.com/watch?v=lJCwezyHNlA).

## 🎲 Enhance Ability and Bardic Inspiration ^skill-support

**Markers:** ⭐⭐ ⚠️ ⏱️ ✨

Use **Enhance Ability** and **Bardic Inspiration** deliberately for important skill checks. This is often better than reloading immediately.

This is especially useful for checks that:

- seem hard to repeat,
- can avoid or simplify a fight,
- open access to resources, vendors, or special items,
- become much easier with the right party member.

More on preparing skill checks is in [Basics and User Interface](01_Basics_and_User_Interface.md).

### 💡 When to Spend Skill Support

| Situation | Spend support? |
|---|---|
| Common repeatable check | Usually no |
| Important dialogue or access check | Usually yes |
| Check can avoid a fight | Yes if the fight would cost more resources |
| Check opens a vendor, item, or route | Yes if the reward is practical |
| Wrong character started the interaction | Consider switching party setup before reloading |

Skill support is most valuable when it prevents a larger resource cost later.

## 💀 Speak with Dead

**Markers:** ⭐ ⏱️ ✨

**Speak with Dead** can be used again after an unhelpful first attempt. A poor first target does not automatically mean the spell is wasted for the day.

Practical value: after a target gives no useful information, do not write off the spell. Other bodies can still provide relevant information.

### 🎯 Practical Use Pattern

If the first body gives no useful result, keep checking likely targets instead of abandoning the spell. Prioritize bodies connected to locked areas, recent fights, or suspicious locations.

The value is not in using the spell everywhere. The value is knowing that one bad target does not end the opportunity. If the wrong character spoke first or no useful questions were asked, try again with a better setup. Disguise, Wild Shape, and other form changes can also matter when a corpse refuses to speak to its killer.

## 🎭 Form Changes and Race-Conditional Gear

**Markers:** ⭐⭐ ⚠️ ✨ 🎯

Form-changing tools such as **Disguise Self** or Mask of the Shapeshifter-style effects can matter beyond dialogue. Some equipment checks the wearer's apparent race or size, and some small openings become easier when a character is in a smaller form.

Practical checks:

- after changing form, re-equip race-conditional gear if the bonus is not appearing,
- use form changes before trying small openings or identity-sensitive interactions,
- remember that disguise tools can help **Speak with Dead** when the target refuses to speak to its killer,
- do not build a whole plan around one form trick without confirming it still works in the current patch.

Source video: [Baldur's Gate 3: 20+ Easily Missed Tips (Spoiler Free) - Proxy Gate Tactician](https://www.youtube.com/watch?v=BGRu99XLSRc&t=16s).

## 🗼 Missable Magic-Item Area

**Markers:** ⭐⭐⭐ ⚠️ ✨

Some areas hide strong magic items behind an optional interaction or secret route. If a place clearly has puzzle mechanics, unusual equipment, or a special ring/key item, check for a follow-up area before leaving.

This is especially relevant for Bless-oriented parties, because one missable item can improve Bless-style strategies. See [Elixirs and Potions](06_Elixirs_and_Potions.md) for [[06_Elixirs_and_Potions#^elixir-of-heroism\|Elixir of Heroism]].

> [!warning]- Spoiler: named area and item
> The **Guiding Light Ring** opens the secret basement of the Arcane Tower. The **Staff of Arcane Blessing** is found there.
>
> The staff synergizes especially well with **[[06_Elixirs_and_Potions#^elixir-of-heroism\|Elixir of Heroism]]**, because [[06_Elixirs_and_Potions#^elixir-of-heroism\|Heroism]] can provide Bless until Long Rest.
>
> ### Why the Basement Matters
>
> | Find | Practical benefit |
> |---|---|
> | Guiding Light Ring | Opens the secret basement |
> | Staff of Arcane Blessing | Improves Bless-oriented strategies |
> | Combination with [[06_Elixirs_and_Potions#^elixir-of-heroism\|Heroism]] | Bless until Long Rest without using a normal Bless spell slot |
>
> The Staff of Arcane Blessing grants **Mystra's Blessing**: under Bless, it improves saving throws and weapon attacks by +1d4 and gives spell attacks a stronger +2d4 bonus. This makes it especially valuable for characters that rely on spell attack rolls.

### ✅ Practical Follow-Up

After finding a key-like item or puzzle reward, do not leave the area without checking whether it opens a hidden follow-up space. The reward may have a clear use case for a specific party plan.

The strongest practical pairing is:

1. Use **[[06_Elixirs_and_Potions#^elixir-of-heroism\|Elixir of Heroism]]** for Bless until Long Rest.
2. Equip or plan around the missable Bless-support item if found.
3. Save concentration for another spell instead of regular Bless.

## 🔮 Plan Spell Resources ^spell-resources

**Markers:** ⭐⭐⭐ ⚔️ ✨

**[[06_Elixirs_and_Potions#^elixir-of-arcane-cultivation\|Elixir of Arcane Cultivation]]** grants additional spell slots. These slots are added, not merely restored.

The combination with **[[06_Elixirs_and_Potions#^potion-of-angelic-slumber\|Potion of Angelic Slumber]]** is especially strong when one caster needs resources restored before a difficult section. Details are in [Elixirs and Potions](06_Elixirs_and_Potions.md).

### 🧠 Resource Planning Before Hard Fights

| Resource | Use case |
|---|---|
| [[06_Elixirs_and_Potions#^elixir-of-arcane-cultivation\|Elixir of Arcane Cultivation]] | More spell slots before a difficult fight |
| [[06_Elixirs_and_Potions#^potion-of-angelic-slumber\|Potion of Angelic Slumber]] | Restore one character outside combat |
| [[06_Elixirs_and_Potions#^elixir-of-heroism\|Elixir of Heroism]] | Long-term Bless without normal concentration cost |
| Scrolls | Emergency access to unprepared spells |

This is most relevant when one caster carries a large part of the party's control, healing, or burst damage.

## 🧠 End Concentration Manually

**Markers:** ⭐⭐⭐ ⚠️ ⚔️ ✨

Concentration spells can remain dangerous after they have served their purpose. If a spell such as a damaging cloud, wall, or terrain control effect is now blocking allies, cancel concentration manually from the character portrait instead of walking the party through it.

Use this habit when:

- a fight has ended but a damaging spell remains,
- allies must cross the controlled area,
- a better concentration spell is about to be cast,
- summons or NPCs might path through the effect.

This is a small UI habit that prevents avoidable damage and accidental hostility.

Source video: [Baldur's Gate 3: 20+ Easily Missed Tips (Spoiler Free) - Proxy Gate Tactician](https://www.youtube.com/watch?v=BGRu99XLSRc&t=501s).


## 🧭 Movement Wizard Utility Kit ^movement-utility

**Markers:** ⭐⭐ ⏱️ ✨ 🎯

A prepared utility caster can solve exploration problems without forcing the main party to carry every niche spell all day. This is useful for locked routes, vertical movement, scouting, or awkward terrain.

| Utility spell or tool | Practical use |
|---|---|
| Feather Fall | Safe drops and fast vertical routing |
| Misty Step | Short teleport for positioning or route access |
| Knock | Opens locks when lockpicking is unavailable or inconvenient |
| Invisibility | Scouting, repositioning, or reaching a waypoint safely |
| Enlarge/Reduce | Access through small openings when the character size allows it |
| Fly / Gaseous Form | Bypass terrain, gaps, bars, or body-blocking problems |
| Remove Curse | Keep a cleanup option available without rebuilding the active party |
| Find Familiar | Lure, scout, or group enemies before a planned opener |
| Speak with Dead + Disguise Self | Try again when a corpse refuses to speak to its killer |

This does not need to be a permanent party slot. The point is to remember that utility magic can be prepared as a tool kit before exploration-heavy sections.

Source: [Baldur's Gate 3: The Ultimate Buff Stacking Guide - SwoleBenji](https://www.youtube.com/watch?v=fjS7vvau0o0&t=1500s).

## 🧰 Small Utility Spells With Big Uses ^utility-spells

**Markers:** ⭐⭐ ⏱️ ✨ 🎯

Some low-level utility spells are stronger when used as tools rather than as direct combat buttons.

| Spell or tool | Practical use |
|---|---|
| Light | Cast it on a cheap object, then throw or place that object to create a movable light source for characters without Darkvision |
| Find Familiar | Scout small routes, distract enemies, lure groups into better positions, or keep the familiar after resting if it was summoned before the Long Rest |
| Mage Hand | Pre-place bombs or objects, shove a ranged enemy from high ground, throw items from its own inventory (see [[#🖐️ Summons as Item Throwers\|Summons as Item Throwers]]), or interact with dangerous positions without risking a party member |
| Silence | Stop spellcasting, mute noisy actions in an area, and protect creatures inside the area from Thunder damage |
| Feather Fall | Cast before vertical routing, risky drops, or fights where falling damage or shove risk matters |

Treat these as preparation tools. The best use is usually before the fight or before the route problem, not after the party is already in a bad position.

Source: [13 Things That are Actually Overpowered in Baldur's Gate 3 - Toyhouze](https://www.youtube.com/watch?v=1g8TJBjPfqw).

## 🖐️ Summons as Item Throwers ^summon-throwers

**Markers:** ⭐⭐ ⚔️ 🎯

Summons have their own turn and their own action. That action is free from the party's perspective, so spending it on item delivery costs nothing that a real character wanted to do.

**Mage Hand** is the clearest example. It has its own inventory: move potions, bombs, or other throwables into it before or during the fight, then let the Hand throw them. Common uses:

| Delivery | Why the Hand is a good carrier |
|---|---|
| Healing potion to a downed or hurt ally | The Hand splashes the heal without a party member spending an action or walking into danger |
| Alchemist's Fire, Smokepowder, or similar bombs | Damage or a hazard surface lands without exposing a real character to opportunity attacks |
| [[06_Elixirs_and_Potions#^potion-of-speed\|Potion of Speed]] onto a grouped party | Buffs the group while every real character still acts normally |

Points to keep in mind:

- The Hand must be summoned **before** it is needed, and items must be transferred into its inventory in advance. Doing all of that mid-fight burns the tempo the trick was supposed to save.
- The Hand is fragile and has low Strength, so treat it as a delivery tool rather than a damage dealer.
- Other summons and familiars can carry and throw items the same way. The principle is general: **the party's throwables should usually be delivered by whichever body has the least valuable action.**

Aiming rules for thrown potions and elixirs are in [[06_Elixirs_and_Potions#^throwing-rules|Throwing Rules]] — in particular, important elixirs should still be drunk, never thrown.

## 🧩 Hidden Spell Interactions ^spell-interactions

**Markers:** ⭐⭐ ⚠️ ✨ 🎯

Some spells have practical secondary uses that are easy to miss. Treat these as tool interactions, not as replacements for the normal spell purpose.

| Spell or setup | Hidden use | Practical note |
|---|---|---|
| Web | Can act as a soft landing surface | Useful when Feather Fall is unavailable and the landing zone can be targeted safely |
| Darkness / Fog Cloud | Obscures an area for stealth actions | Strong for line-of-sight control; avoid using it where it blocks your own ranged plan |
| Summon spells | Summon outside combat, then prepare a different spell | The summon can persist while the caster frees the prepared-spell slot for the next section |
| Create or Destroy Water | Can reveal an invisible target if the area is guessed correctly | Best when the invisible creature's likely position is constrained |
| Sanctuary | Protects a revived or healed ally after a rescue turn | Healing Word plus Sanctuary is a strong recovery pattern because Sanctuary is a bonus action |
| Enlarge/Reduce + disguise/form tools | Can help with small routes and throwing ally setups | Confirm the route and form size before treating it as the main plan |
| Grease + fire | Can extend a small fire surface into a larger hazard | Keep allies clear; this is area control as much as damage |
| Ray of Frost / ice tools | Can put out fire and leave a water/ice setup | Useful for removing fire, then setting up lightning or freezing follow-up |

Sanctuary is especially useful on a support caster who will heal, summon, or maintain concentration without directly harming enemies. It does not make the character immune to falling or area hazards, so positioning still matters.

Source: [9 Hidden Spell Interactions in Baldur's Gate 3 - Toyhouze](https://www.youtube.com/watch?v=XOpwzSepoBk).

## 🛡️ Weapon Bond for Valuable Weapons ^weapon-bond

**Markers:** ⭐⭐ ⚠️ ⚔️ 🎯

An Eldritch Knight can use **Weapon Bond** on an important weapon, then return that weapon to the character who actually uses it. This can protect valuable weapons from disarm-heavy enemies and can also make thrown weapons return without requiring a specific returning weapon.

Use this for weapons that would be costly to drop, lose, or spend actions recovering. Treat it as an advanced setup trick and recheck it after changing equipment or resting.

Source: [Baldur's Gate 3: The Ultimate Buff Stacking Guide - SwoleBenji](https://www.youtube.com/watch?v=fjS7vvau0o0&t=1104s).

## 👁️ See Invisibility Without an Elixir Slot

**Markers:** ⭐⭐ ⏱️ ✨

A **Scroll of See Invisibility** can provide long-duration invisible-enemy detection without consuming the character's elixir slot. This is useful when the elixir slot is reserved for Strength, Bloodlust, Vigilance, Heroism, or another core plan.

Source: [Baldur's Gate 3: The Ultimate Buff Stacking Guide - SwoleBenji](https://www.youtube.com/watch?v=fjS7vvau0o0&t=1471s).

## ✅ Short Magic Checklist ^magic-checklist

**Markers:** ⭐⭐ ✨

| Situation | Check |
|---|---|
| Important skill check | Use Enhance Ability or Bardic Inspiration |
| New Wizard spell | Check Learn More Spells and cost |
| Exploration-heavy section | Prepare a movement utility caster before changing areas |
| Valuable weapon may be disarmed | Consider Weapon Bond setup before the fight |
| Dead NPC gives no value | Use Speak with Dead on another target |
| Missable magic area completed | Check whether a key-like item opens a follow-up space |
| Hard fight for casters | Plan [[06_Elixirs_and_Potions#^elixir-of-arcane-cultivation\|Arcane Cultivation]] and [[06_Elixirs_and_Potions#^potion-of-angelic-slumber\|Angelic Slumber]] |
